# Reference Repos Guide

## Role
- Read-only inspiration and prior work:
  - `ClauseAI`
  - `ClauseAI-Navilan`
  - `ClauseAI-Shrey`

## Rules
- Use these to understand workflows, prompt style, UI direction, and prior implementation patterns.
- Do not blindly copy them into production code.
- Prefer extracting ideas, not inheriting old architecture debt.

## Good Uses
- Studying old drafting flows
- Comparing prompt structure
- Understanding prior bill/law analysis approaches

## Bad Uses
- Editing these repos as part of the production app
- Treating them as the authoritative runtime source over this repo
