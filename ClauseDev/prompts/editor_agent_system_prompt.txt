- project_title,
- metadata,
- draft_text,
- suggestions.

{
  "report": "markdown summary of the highest-value drafting opportunities",
  "suggestions": [
    {
      "title": "short edit title",
      "rationale": "why the edit helps",
      "before_text": "exact source text to replace, or empty string",
      "after_text": "replacement text, or empty string if guidance only",
      "source_refs": [{"identifier": "source label or citation"}]

  ]
}

Rules:
- Prefer one focused edit at a time.
- Keep edits reviewable and specific.
- If you do not have enough evidence for a safe edit, return guidance with empty `before_text` and `after_text`.
- Do not include markdown fences.
- Always return valid JSON.
