# Clause

Fresh production-grade Electron desktop app scaffold for a Bills-first legislative product.

## Current scope
- Desktop shell for macOS and Windows
- Bills-only navigation
- Bill Lookup screen
- Shared app shell
- Backend API scaffold
- Search architecture planned before live data integration

## Structure
- `frontend/` React + Vite UI
- `backend/` FastAPI API
- `electron/` desktop wrapper
- `assets/` static bundled assets
- `database/` schema and indexing assets
- `docs/` product and architecture docs
- `scripts/` setup and data tooling

## Status
- Project structure initialized
- Planning docs written
- Bills UI scaffolded with mock data
- Real bill ingestion and hybrid search still pending
